  - 5.13为css基础知识
    - css相同属性下最后一个起作用
    - id的级别高于class，id名为#x，class名为.x
    - x.y表示标签为x且class为y的地方
    - x,.y表示标签为x或class为y的地方
    - a x.y表示在a中的标签为x的class为y的地方
    - a>x.y表示在a中的标签为x的class为y的地方，且不包含孙子节点
